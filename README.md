# UIDE Website 2025

Sitio web moderno y atractivo para la Universidad Internacional del Ecuador (UIDE), desarrollado con React, TypeScript, Tailwind CSS y Framer Motion.

## Características

- 🎨 Diseño moderno y responsivo
- ⚡ Rendimiento optimizado
- 🎭 Animaciones suaves con Framer Motion
- 📱 Adaptable a todos los dispositivos
- 🎯 SEO optimizado
- 🔍 Accesibilidad mejorada

## Tecnologías

- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- Vite
- React Router
- Heroicons

## Requisitos

- Node.js 18 o superior
- npm 9 o superior

## Instalación

1. Clonar el repositorio:
```bash
git clone https://github.com/uide/website-2025.git
cd website-2025
```

2. Instalar dependencias:
```bash
npm install
```

3. Iniciar el servidor de desarrollo:
```bash
npm run dev
```

4. Abrir [http://localhost:5173](http://localhost:5173) en tu navegador.

## Scripts Disponibles

- `npm run dev` - Inicia el servidor de desarrollo
- `npm run build` - Construye la aplicación para producción
- `npm run preview` - Vista previa de la versión de producción
- `npm run lint` - Ejecuta el linter
